# Vanigan.org Business Directory

A complete, production-grade business directory platform for Tamil Nadu built with React.js and Vite.

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern UI/UX**: Clean, professional light theme with smooth animations.
- **Business Directory**: Interactive listing page with search, filters, and pagination.
- **Multi-Step Form**: Professional "List Your Business" flow with progress tracking.
- **SEO Optimized**: Dynamic page titles and meta descriptions using `react-helmet-async`.
- **Animations**: Subtle, high-performance reveal animations using `framer-motion`.

## 🛠️ Tech Stack

- **Framework**: React.js 18+
- **Build Tool**: Vite
- **Routing**: React Router v6
- **Animations**: Framer Motion
- **Icons**: React Icons (Fa, Fa6)
- **SEO**: React Helmet Async
- **Styling**: Vanilla CSS with Variables

## 📦 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Shared UI components (Navbar, Footer, Cards, etc.)
- `src/pages`: Individual page components (Home, About, BusinessList, etc.)
- `src/data`: Sample business and category datasets
- `src/styles`: Global CSS and variable definitions
- `src/hooks`: Custom React hooks for animations
